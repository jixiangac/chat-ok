## ADDED Requirements

### Requirement: 步骤按钮状态控制

创建任务流程中每个步骤的"下一步/创建"按钮 **MUST** 在配置未完成时禁用。

#### Scenario: 周期设定步禁用条件

**Given** 用户在周期设定步骤  
**When** 总天数小于周期天数，或周期天数为 0  
**Then** "下一步"按钮显示为禁用状态  
**And** 按钮呈现灰色，不可点击

#### Scenario: 周期设定步启用条件

**Given** 用户在周期设定步骤  
**When** 总天数大于等于周期天数，且周期天数大于 0  
**Then** "下一步"按钮显示为启用状态  
**And** 按钮使用主题色，可点击

#### Scenario: 类型选择步禁用条件

**Given** 用户在类型选择步骤  
**When** 未选择任何任务类型  
**Then** "下一步"按钮显示为禁用状态

#### Scenario: 配置步数值型任务禁用条件

**Given** 用户在配置步骤且选择了数值型任务  
**When** 任务标题为空，或起始值为空，或目标值为空  
**Then** "创建任务"按钮显示为禁用状态

#### Scenario: 配置步清单型任务禁用条件

**Given** 用户在配置步骤且选择了清单型任务  
**When** 任务标题为空，或清单总数为空/无效  
**Then** "创建任务"按钮显示为禁用状态

#### Scenario: 配置步打卡型任务禁用条件

**Given** 用户在配置步骤且选择了打卡型任务  
**When** 任务标题为空  
**Then** "创建任务"按钮显示为禁用状态

#### Scenario: 配置完成后启用创建

**Given** 用户在配置步骤  
**When** 所有必填项均已填写且有效  
**Then** "创建任务"按钮显示为启用状态  
**And** 用户可以点击创建任务
