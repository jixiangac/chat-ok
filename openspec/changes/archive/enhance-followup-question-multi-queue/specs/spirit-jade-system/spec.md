## ADDED Requirements

### Requirement: 创建任务灵玉消耗计算

创建任务时的灵玉消耗 **MUST** 基于任务完成后可获得的灵玉数量动态计算。

#### Scenario: 主线任务消耗计算

**Given** 用户创建主线任务  
**And** 100%完成该任务可获得 N 灵玉  
**When** 计算创建消耗  
**Then** 消耗 = N × 0.2，向下取整到十位（如 511→510，655→650）  
**And** 消耗最低为 500  
**And** 消耗最高为 1000

#### Scenario: 支线任务消耗计算

**Given** 用户创建支线任务  
**And** 100%完成该任务可获得 N 灵玉  
**When** 计算创建消耗  
**Then** 消耗 = N × 0.15，向下取整到十位  
**And** 消耗最低为 200  
**And** 消耗最高为 500

#### Scenario: 消耗取整示例

**Given** 计算得出的消耗值为 511  
**When** 应用取整规则  
**Then** 最终消耗为 510

#### Scenario: 消耗下限保护

**Given** 计算得出的主线任务消耗值为 300  
**When** 应用范围限制  
**Then** 最终消耗为 500（下限）

#### Scenario: 消耗上限保护

**Given** 计算得出的主线任务消耗值为 1200  
**When** 应用范围限制  
**Then** 最终消耗为 1000（上限）
